# FinanceApp
Простое консольное Java-приложение для управления личными финансами.
Созданное как итоговое задания по дисциплины ООП.

## Основные возможности

-  Много пользователей — авторизация и регистрация по логину/паролю.
-  Кошелёк с балансом, доходами, расходами и операциями по категориям.
-  Бюджеты по категориям.
-  Отчёты по периодам.
-  Экспорт и импорт всех операций в формат `CSV`.
-  Уведомления при превышении лимита бюджета и отрицательном балансе.
-  Управление категориями: установка, изменение, переименование, удаление.
-  Справка командой `help`.

---

## Запуск через IDE

1. Открой проект в IntelliJ IDEA или VS Code.
2. Укажи Main.java как точку входа.
3. Запусти программу (Run → Run 'Main').
